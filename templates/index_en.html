📖 Usage Guide

Supported Scenarios:

  • RVM Full: "full", "filled", "need cleaning"
  • Machine Fault: "fault", "broken", "not working"
  • Inquiry: "inquiry", "question", "location", "hours"
  • Complaint: "complaint", "dissatisfied", "problem"
  • Praise: "praise", "thank", "excellent"

Integration Example:

// JavaScript Integration
fetch('/api/auto-reply', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        phone_number: '+60123456789',
        message: 'RVM is full',
        location: 'Dataran Banting',
        rvm_id: 'RVM-BANTING-001'
    })
})
.then(response => response.json())
.then(data => {
    console.log('Response:', data.response);
    // Send this response back to customer
});

Sample Responses:

RVM Full: "Thank you for notifying that the RVM at Dataran Banting is full. Our cleaning team will arrive within 2 hours..."

Machine Fault: "Sorry for the inconvenience. The RVM at Puchong has a malfunction. Our technician will arrive within 1 hour..."

Inquiry: "Thank you for your inquiry. Our RVM locations: 1. Dataran Banting, 2. Puchong, 3. Subang Jaya..."
