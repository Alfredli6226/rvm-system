📖 使用说明

支持的场景:

  • RVM已满: "满", "满了", "需要清理"
  • 机器故障: "故障", "坏了", "不能用"
  • 咨询: "咨询", "问", "位置", "时间"
  • 投诉: "投诉", "不满意", "问题"
  • 表扬: "表扬", "谢谢", "很好"

集成方式:

// JavaScript 示例
fetch('/api/auto-reply', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
        phone_number: '+60123456789',
        message: 'RVM已满，请清理',
        location: 'Dataran Banting',
        rvm_id: 'RVM-BANTING-001'
    })
})
.then(response => response.json())
.then(data => console.log(data));
